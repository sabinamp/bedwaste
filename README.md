# bedwaste
## Android App with Java
This is the prototype of an Android application for travellers that need to find last minute hotel accommodation deals. 
Technology stack:
* Android with Java,
* the Retrofit library for remote calls to web services
* GSON to serialize Java objects to JSON and vice versa.
* Restful API to a server side hotel service stub, which provides hard coded data as the information on hotel deals in ZÃ¼rich. 
* Google Maps Android API 
* Android Picasso library -an image loading library for Android.
